## Development Setup

```bash
# Clone repository
git clone https://github.com/YourUsername/gemini-tray.git
cd gemini-tray

# Install dependencies
npm install

# Start development
npm start

# Build
npm run build:win  # Windows
npm run build:linux  # Linux
```
