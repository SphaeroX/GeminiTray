# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** post it publicly as an issue
2. Send an email to: [security@example.com]
3. Wait for a response (usually within 48 hours)

## Security Measures

- The app uses Electron with security features enabled
- No storage of sensitive data
- No automatic updates (you have full control)

## Known Limitations

- This is a community project without professional security audits
- Use at your own risk
